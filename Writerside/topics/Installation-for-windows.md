# Installation for Windows

## What you will need:
1. WSL 2.0
2. a copy of Git
3. At least 5GB of storage
4. Mysql
> This beta must be used with a linux distro that supports apt-get
> 
{style="warning"}

First go to Turn Windows features on or off and check wsl
> Your computer may restart
>
{style="warning"}

Then Go to the Windows store and install an apt-get system(Like Kali Linux, Ubuntu or Debian)
> This tutorial will use Kali Linux
> 

Open the apt-get system and install git with this command :
``` sudo apt-get install git```
then install the gui with this command :
For Kali:
``` git clone http://localhost:3000/caodial/Foxkit.git ```
